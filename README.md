# 🎬 Movie Finder

A clean, responsive single-page web application built with **React** to explore the vast world of cinema. Search for any movie, get detailed info, rate your favorites, and manage your personal watchlist—all stored locally in your browser.

## 🚀 Key Features

- **Smart Search:** Search through a massive movie database powered by IMDb API.
- **Detailed View:** Get in-depth information about any movie you select.
- **Personal Rating:** Rate your favorite movies on a scale of 1 to 10.
- **Watched List:** Keep track of movies you've watched with a personalized summary (Runtime, Your Rating, IMDb Rating).
- **Persistence:** Your data is saved via `localStorage`, so your list remains intact even after closing the browser.
- **Clean UI:** Minimalist and intuitive interface built with React.

---

## 🛠 Tech Stack

- **Library:** React
- **Language:** JavaScript (ES6+)
- **Storage:** LocalStorage API
- **API:** IMDb/OMDb API (External)

---

## 📋 How It Works

1.  **Search:** Use the search bar to find any movie.
2.  **Explore:** Click on a movie result to view detailed information.
3.  **Rate:** Give the movie your own rating (1-10).
4.  **Watchlist:** Add the movie to your watched list to keep track of your progress.
